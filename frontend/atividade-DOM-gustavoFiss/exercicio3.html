Exercício 3

Mostrar / Ocultar

Alterna style.display de cada parágrafo entre "none" e "block" a cada clique.

Primeiro parágrafo — visível.

Segundo parágrafo — visível.

Terceiro parágrafo — visível.

Quarto parágrafo — visível.

Estado: visível
Código
function alternarVisibilidade() { const paragrafos = document.getElementsByTagName('p'); for (let i = 0; i < paragrafos.length; i++) { if (paragrafos[i].style.display === 'none') { paragrafos[i].style.display = 'block'; // torna visível } else { paragrafos[i].style.display = 'none'; // oculta } } }
