Exercício 4

Trocar Cor de Fundo

Usa getElementById e um addEventListener para aplicar a cor digitada à div.

—
colorBox
Código
// Seleciona os elementos pelo id const colorBox = document.getElementById('colorBox'); const corInput = document.getElementById('corInput'); const btnAplicar = document.getElementById('btnAplicar'); // addEventListener dispara ao clique no botão btnAplicar.addEventListener('click', function () { const cor = corInput.value.trim(); if (cor) { colorBox.style.backgroundColor = cor; } });
