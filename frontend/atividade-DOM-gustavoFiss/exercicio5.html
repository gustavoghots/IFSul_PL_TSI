Exercício 5

Esconder / Mostrar

Alterna style.display de um parágrafo com id="hiddenText" usando addEventListener.

Este parágrafo tem id="hiddenText". Clique no botão abaixo para escondê-lo ou exibi-lo novamente.

display: block
Código
const paragrafo = document.getElementById('hiddenText'); const btn = document.getElementById('btnToggle'); btn.addEventListener('click', function () { if (paragrafo.style.display === 'none') { paragrafo.style.display = 'block'; // mostra } else { paragrafo.style.display = 'none'; // esconde } });
