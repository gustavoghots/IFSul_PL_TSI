Exercício 6

Aumentar / Diminuir Fonte

Modifica style.fontSize progressivamente ao clicar nos botões.

O tamanho desta fonte muda progressivamente a cada clique nos botões ao lado. Use os controles para aumentar ou diminuir o texto conforme desejar.

16
px atual
Código
const paragrafo = document.getElementById('textoParagrafo'); const btnAumentar = document.getElementById('btnAumentar'); const btnDiminuir = document.getElementById('btnDiminuir'); let tamanhoAtual = 16; // tamanho inicial em px btnAumentar.addEventListener('click', function () { tamanhoAtual += 2; paragrafo.style.fontSize = tamanhoAtual + 'px'; }); btnDiminuir.addEventListener('click', function () { if (tamanhoAtual > 8) { // limite mínimo tamanhoAtual -= 2; paragrafo.style.fontSize = tamanhoAtual + 'px'; } });
