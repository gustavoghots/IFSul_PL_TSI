Exercício 7

Interatividade com o Mouse

Usa mouseover e mouseout para alterar a cor de fundo de uma <div> ao passar o mouse.

Passe o mouse aqui
Último evento
—
Cor de fundo atual
var(--surface-3)
Código
const caixa = document.getElementById('mouseBox'); // mouseover — dispara quando o mouse entra na div caixa.addEventListener('mouseover', function () { caixa.style.backgroundColor = '#4568f5'; }); // mouseout — dispara quando o mouse sai da div caixa.addEventListener('mouseout', function () { caixa.style.backgroundColor = ''; // volta ao valor original });
