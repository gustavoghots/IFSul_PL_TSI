Grupo D

Seletores de Atributo

Selecionar elementos pela presença ou valor de um atributo HTML.

Exercício 12 Seletor a[target] — fundo rosa para <a> que possuem qualquer atributo target.
IFSul — sem target (sem estilo). Pelotas — target="_blank" (rosa). Wikipedia — target="_top" (rosa).
/* Exercício 12 — [target] presença do atributo */ a[target] { background-color: pink; }
Exercício 13 Seletor a[target="_blank"] — fundo azul somente para <a> com target="_blank".
IFSul — sem target (sem estilo). Pelotas — target="_blank" (azul). Wikipedia — target="_top" (sem estilo).
/* Exercício 13 — [target="_blank"] valor exato */ a[target="_blank"] { background-color: blue; color: #fff; }
