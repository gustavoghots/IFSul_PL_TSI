Grupo E

Atributo com Padrão

Seletores [attr*=], [attr^=], [attr$=] e [attr~=] aplicados a imagens.

Exercício 14 Seletor img[title*="carta"] — borda vermelha em imagens cujo title contém a palavra "carta".
title="coringa carta" ✓
title="sequencia de cartas" ✓
title="ases em sequencia…" ✓
/* Exercício 14 — [title*="carta"] contém */ img[title*="carta"] { border: 4px solid red; }
Exercício 15 Seletor img[title^="sequencia"] — borda azul em imagens cujo title começa com "sequencia".
title="coringa carta" ✗
title="sequencia de cartas" ✓
title="ases em sequencia…" ✗
/* Exercício 15 — [title^="sequencia"] começa com */ img[title^="sequencia"] { border: 4px solid blue; }
Exercício 16 Seletor img[title$="carta"] — borda verde em imagens cujo title termina com "carta".
title="coringa carta" ✓
title="sequencia de cartas" ✗
title="ases… cartas " ✗ (espaço)
/* Exercício 16 — [title$="carta"] termina com */ img[title$="carta"] { border: 4px solid green; }
Exercício 17 Seletor img[title~="flor"] — borda laranja em imagens cujo title contém a palavra exata "flor". Nenhuma das imagens do exemplo contém "flor", então foi adicionada uma de teste.
title="coringa carta" ✗
title="sequencia de cartas" ✗
title="baralho de flor especial" ✓
/* Exercício 17 — [title~="flor"] palavra exata */ img[title~="flor"] { border: 4px solid orange; }
